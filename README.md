## bin-lookup
An Android app written in Java that allows users to lookup data associated with a credit/debit card's BIN/IIN via [binlist.net's API](https://binlist.net/).


## Libraries/frameworks used
[Retrofit2](https://github.com/square/retrofit)  
[Moshi](https://github.com/square/moshi)  
[OkHttp's Logging interceptor](https://github.com/square/okhttp/tree/master/okhttp-logging-interceptor)  
[Android Jetpack](https://developer.android.com/jetpack)  
[Dagger](https://github.com/google/dagger)  
[RxJava2](https://github.com/ReactiveX/RxJava)
